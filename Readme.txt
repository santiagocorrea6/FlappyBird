FLAPPYBIRD

El presente juego fue desarrollado para el curso de procesamiento digital de  imágenes ofrecido 
por la Universidad  de  Antioquia,  el programa emula el famoso juego  multiplataforma conocido 
como  FlappyBird  que  consiste  en  un  desplazamiento  lateral  donde  con  reconocimiento de 
imágenes el jugador controla a un pájaro,  intentando  volar entre columnas de tubos verdes sin 
golpearlos.

Requerimientos:
  -	Matlab versión 2013 o posteriores.
  -	Cámara web integrada.
  -	Objeto de color amarillo de un diámetro mínimo de 2cm

Ejecución: El proyecto  cuenta con una serie de funciones, imágenes y sonidos; los cuales deben
estar en la misma carpeta del  presente  documento.  Para la  ejecución del programa se abre el 
archivo flappybird.m con Matlab, seleccionar la pestaña Editor y click en Run, luego de esto el 
juego iniciara  su  ejecución.  Para controlar el vuelo del pájaro  es necesario usar un objeto 
de color amarillo preferiblemente circular y de un diámetro mínimo de 2 cm.
